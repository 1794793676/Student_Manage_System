# 学生选课管理系统实验说明

## 实验目的

完成一个可运行、可演示、结构清晰的学生选课管理系统，理解 MVC 分层架构、面向对象建模、Session 登录控制、数据库持久化和基础业务规则校验。

## 实验环境

- JDK：Java 17
- 构建工具：Maven
- 后端框架：Spring Boot 3.2.5
- 页面模板：Thymeleaf
- 数据库：H2 内存数据库
- 浏览器：Chrome、Edge 或其他现代浏览器

## 系统功能

学生功能：

- 登录和退出登录
- 查看课程列表
- 按课程名称或教师搜索课程
- 选择课程
- 退选课程
- 查看已选课程

管理员功能：

- 登录和退出登录
- 查看课程列表
- 添加课程
- 修改课程
- 删除课程
- 查看每门课程已选人数
- 查看学生选课记录

## 系统模块

- 登录管理模块：负责账号校验、Session 保存和退出登录。
- 学生选课模块：负责课程查询、选课、退课、已选课程查询。
- 管理员课程模块：负责课程新增、修改、删除和课程列表展示。
- 权限拦截模块：通过拦截器限制未登录用户和角色不匹配用户访问。
- 数据初始化模块：项目启动时自动初始化测试用户、学生、课程和选课记录。

## 数据库设计

### users 用户表

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| username | 用户名 |
| password | 密码 |
| role | 角色，student 或 admin |

### students 学生表

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| student_no | 学号 |
| name | 姓名 |
| major | 专业 |
| grade | 年级 |
| user_id | 关联用户 |

### courses 课程表

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| course_name | 课程名称 |
| teacher | 授课教师 |
| credit | 学分 |
| capacity | 课程容量 |
| selected_count | 已选人数 |

### student_courses 学生选课关系表

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| student_id | 学生 ID |
| course_id | 课程 ID |
| select_time | 选课时间 |

## 页面说明

- `/login`：输入用户名和密码，根据角色进入学生或管理员首页。
- `/student/home`：学生功能入口。
- `/student/courses`：课程列表、课程搜索和选课操作。
- `/student/selected`：查看已选课程和退课操作。
- `/admin/home`：管理员功能入口。
- `/admin/courses`：课程增删改查和已选人数展示。
- `/admin/selections`：学生选课记录展示。

## 运行方法

```bash
mvn clean package
mvn spring-boot:run
```

浏览器访问 `http://localhost:8080/login`。
